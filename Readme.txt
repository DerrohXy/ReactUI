ReactUI is a javascript library providing custom designed React components to make the building of user interfaces with react easier.
It provides numerous components out of the box including an Application object providing browser history navigation capabilities 
using the window.location.hash object. The js file bundles with a css file which provides styling to the custom components.

A global V1 object provides the functions, components and objects necessary.

    Application,
    AudioView,

    HorizontalLayout,
    HorizontalScrollWindow,
    ImageButton,
    ImageView,

    Link,
